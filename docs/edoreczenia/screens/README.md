# Screeny panelu e-Doręczeń (biznes.gov.pl) — i-Future sp. z o.o.

Materiał referencyjny do budowy instrukcji w dialogu integracji e-Doręczeń w Concertivo.

- `01-skrzynka-odebrane.png` — widok skrzynki e-Doręczeń (Strona główna → Moje konto → e-Doręczenia → AE:PL-...). Pokazuje: adres do doręczeń, rolę użytkownika (Administrator), foldery (Odebrane / Wysłane / Robocze / Usunięte / Foldery / Wątki) oraz menu „Ustawienia / Uprawnienia / Usługi dodatkowe / Subskrypcje / Reguły wiadomości".
- `02-uprawnienia-systemy.png` — Uprawnienia w skrzynce do e-Doręczeń → zakładka **Systemy** z przyciskiem **Dodaj system**. To miejsce, w którym rejestruje się Concertivo jako zewnętrzny system integrujący się ze skrzynką.
- `03-dodaj-system.png` — formularz **Dodaj system**: Nazwa systemu, Opis (opcjonalnie), upload pliku certyfikatu (.crt / .cer / .pem, do 10 MB) — „kwalifikowany środek uwierzytelniający" (publiczna część certyfikatu Concertivo).

Wykorzystać w instrukcji krok-po-kroku w komponencie EdoreczeniaSetupInstructions (do utworzenia w fazie 1).
